### 설명
카테고리 코드 PRODUCT_CODE 별 상품 개수를 작성하려고 한다.    
카테고리 앞 2자리를 보면 A2, A3, B2로 되어 있고    
이것은 부분문자열을 가져오는 함수는 SUBSTR을 사용하여     
PRODUCT_CODE의 첫 번째부터 길이가 2자인 문자열을 가져온다.
