### 설명
생일이 3월인 여성 회원의 ID, 이름, 성별, 생년월일을 조회하려고 한다.    
출력할 때 생년월일은 TO_CHAR을 사용하여 연, 월, 일까지만 출력하게 하고    
조건문에는 MM이 03이면서 NULL이 아니고 GENDER가 W인 것을 선택한다.
