### 설명
테이블에 NULL이 있으면 연산이 산술연산이나 비교연산이 불가능해진다.               
NULL은 NVL 함수를 사용하여 NULL인 경우 대체되는 데이터를 넣을 수 있다.            
NVL 함수를 사용하여 name에서 NULL인 곳에 No name을 저장한다.
