### 설명
만원을 단위로 만원대, 2만원대, 3만원대의 가격의 카운트를 출력한다.          
만원대는 만단위에서 내림(FLOOR)을 하여 구할 수 있다.    
이중쿼리를 사용하여 만 아래의 값을 모두 제거한다.   
현재 SQL문처럼 나눴다가 곱해도 되지만 FLOOR(PRICE, -5)를 사용하여 만 단위까지 출력할 수도 있다.
