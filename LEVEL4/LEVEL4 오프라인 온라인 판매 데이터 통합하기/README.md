### 설명
두 개의 테이블의 컬럼을 합치는 것을 JOIN이라고 한다. 서로 가지고 있는 컬럼이 유사할 때        
위아래로 붙이는 것을 UNION이라고 한다.        
컬럼의 이름을 같게 명시한 다음 UNION을 사용하여 위아래로 연결하였고 OFFLINE_SALE 테이블에는 USER_ID가 없기 때문에        
이 값은 모두 NULL로 처리했다.
