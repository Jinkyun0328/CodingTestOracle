### 설명
INS에서는 중성화 되지 않았으면서 OUTS에서는 중성화된 동물의 데이터를 조회하려고 한다.   
두 테이블에서 데이터를 가져오면서 두 테이블에 모두 데이터가 있어야 하므로 INNER JOIN으로 연결한다.   
중성화 되지 않은 동물은 Intact로 시작하고 중성화된 동물은 Spayed나 Neutered가 포함된다.
