### 설명
CART_ID가 같으면 같은 장바구니에 담겨있는 것이다.         
동일한 CART_ID에 요거트와 우유가 동시에 들어있는 CART_ID를 조회하려고 한다.         
같은 테이블을 INNER JOIN으로 연결하여 조회하고         
이름이 요거트이면서 동시에 우유인 튜플을 조회한다.
